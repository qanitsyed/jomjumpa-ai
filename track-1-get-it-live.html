⭐ THE GOLDEN RULE
Always end your prompt with: "Build this as a single self-contained HTML file with no external libraries. Clean, modern design. Mobile-friendly."

SOMETHING NOT WORKING?

Tap whatever sounds like your problem.

Most issues are tiny and have a one-line fix. Find yours below.

🤖
My AI keeps using React or strange imports
▸

This means it ignored the Golden Rule. Send this exact message:

Rewrite this as a single plain HTML file with all CSS and JavaScript inline. No imports, no React, no external libraries.
📥
How do I get the code out — download or copy?
▸

Download is the easy path. Look for a download icon near the code block.

💻 On a laptop (Claude or ChatGPT web): Hover over the code block — a Download button appears. Tap it. The file lands in your Downloads folder.

📱 Claude mobile app: Tap the three dots (⋯) at the top-right of the artifact preview. Pick Download (or Copy, if you prefer).

📱 ChatGPT mobile app: Tap the download icon below the code block. (The mobile ChatGPT app doesn't make Copy easy — Download is your route.)

If download doesn't work for any reason, copy the code and use the paste-route via EdgeOne — see the "I'm on my phone" troubleshoot below.

💾
Where's my file? Do I need to rename it?
▸

Try downloading first. Your file is in your Downloads folder. The AI usually names it index.html already — perfect.

If it's named something else (like document.html or app.html), just rename it to index.html. No need to open it or edit anything inside.

If download doesn't work at all: copy the code from your AI, open pages.edgeone.ai/drop, tap Paste Code, paste, and submit. Same outcome, different route.

📱
I'm on my phone — full mobile flow
▸

You've got two paths. Try downloading first.

Path A — Download & upload (try first):

1. Download the file from your AI (see "How do I get the code out?" above)
2. Open your phone's Files app — look in the Downloads folder. Your file should be there.
3. Open netlify.com/drop in your phone browser → tap to choose the file → upload. Get a URL.

Path B — Copy & paste (if you can't find or upload the file, or you're on iPhone):

1. Copy the code from your AI (Claude mobile: ⋯ → Copy)
2. Open pages.edgeone.ai/drop
3. Tap Paste Code → paste → submit. Get a URL.

⚠️
Netlify says error / nothing loads
▸

The most common cause: your file isn't named exactly index.html. Even Index.html or index.html.txt will break it.

Check the file name in your Downloads folder. Lowercase, ends in .html, no extra extensions. Rename if needed and try uploading again.

If that's right and the page still doesn't load — wait 30 seconds and refresh. Netlify sometimes takes a moment.

PREFER A FULL WALKTHROUGH?

The slow, step-by-step version.

i.
Pick a small idea
Keep it tiny. A calculator, a tracker, a tool you'd actually use. The smaller the idea, the faster the win.
ii.
Open your AI
Go to claude.ai or chat.openai.com. Both work for this.
iii.
Type your prompt
Describe your idea in plain English. End with the Golden Rule (it's at the top of this page). Send.
iv.
Check the preview
The AI should show a working preview. Try it. Does it do roughly what you wanted? If not, tell it what to change. Don't overthink — keep iterating.
v.
Download the code file
Look for a Download button or icon near the code block (web: hover over the code; Claude mobile: ⋯ menu on artifact preview; ChatGPT mobile: download icon below code). The file goes to your Downloads folder.
vi.
Check the file is named index.html
Open your Downloads folder. The file is usually already named index.html. If not, rename it. No need to open it or edit anything inside — your code is already there.
vii.
Upload it to Netlify
Go to netlify.com/drop. Drag your file onto the page (or tap to choose it on mobile). Wait 10 seconds. You'll see a live URL.

📱 Phone can't upload the file? Go back, copy the code from your AI, open pages.edgeone.ai/drop, tap Paste Code, paste, and submit.
viii.
Share it
Open your URL. It works on the internet now. Anyone with the link can visit. Paste it into the form to add it to the Wall of Apps. That's the whole thing.
